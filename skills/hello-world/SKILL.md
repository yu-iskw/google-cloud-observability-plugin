# Hello World

Use this skill as a minimal example for plugin skill authoring.

## Purpose

Provide a deterministic, low-risk skill that can be used to verify skill discovery and execution wiring.

## Inputs

- Optional greeting target (defaults to `world`).

## Behavior

1. Read the user request.
2. If a target is provided, greet that target.
3. Otherwise respond with `Hello, world!`.

## Output

A short greeting message.
